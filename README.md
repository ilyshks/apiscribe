# APIScribe 🌟

**Автоматическое документирование API через анализ трафика.**

APIScribe — это прозрачный HTTP/HTTPS прокси-сервер, который перехватывает и анализирует трафик API для автоматической генерации документации в формате OpenAPI/Swagger. **Без необходимости писать аннотации в коде!**

## 🚀 Статус проекта

Проект находится в **активной разработке**.
